Creé este sitio usando <CustomLink href="https://nextjs.org/">Next.js</CustomLink>
un framework de
<CustomLink href="https://reactjs.org/">React</CustomLink> y lo publiqué en
<CustomLink href="https://vercel.com/about/">Vercel</CustomLink>. El código está
disponible en <CustomLink href="https://github.com/tjcocucci/tadeojcocucci-vercel-app/"> este repositorio</CustomLink>.

El contenido está escrito en <CustomLink href="https://mdxjs.com/">MDX</CustomLink> y se embebe como componentes de React mediante el uso de y <CustomLink href="https://contentlayer.dev/">Contentlayer</CustomLink>.

Para estilizar el sitio, utilicé <CustomLink href="https://tailwindcss.com/">Tailwind CSS</CustomLink> e implementé el modo oscuro y claro combinando la funcionalidad de Tailwind con Context de React. Las utilidades de Tailwind también me permitieron crear un diseño responsivo. 

El sitio admite dos idiomas (español e inglés), lo cual lo logré mediante el uso de <CustomLink href="https://www.i18next.com/">i18next</CustomLink> y nuevamente un Context Provider. La URL prepende el código del idioma actual, lo cual se logra personalizando el `middleware.ts` de Next.js.

<CustomLink href="/map"> Aquí</CustomLink> creé un mapa de la República Argentina con celdas de Voronoi para seccionar el territorio en relación a los aeropuertos del país. El mapa está hecho con <CustomLink href="https://d3js.org/">D3.js</CustomLink> y los datos provienen de <CustomLink href="datos.gob.ar">datos.gob.ar</CustomLink>
