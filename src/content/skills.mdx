Mi investigación de doctorado involucró modelado basado en agentes, asimilación de datos y machine learning. Este trabajo involucró usar python para modelar en combinación con C para las tareas intensivas en la CPU. Completé cursos de posgrado en aprendizaje automático, computación paralelay estadística, entre otros. Como profesor universitario superviso laboratorios de programación que incluyen: Sistemas Operativos con C y Paradigmas de la Programación con Haskell, Java y Hadoop. También he estudiado una variedad de técnicas de computación paralela: SIMD intrinsics, paralelismo de hilos on openMP y programación para GPU con CUDA. Adicionalmente utilizo tecnologías como Docker y git.
